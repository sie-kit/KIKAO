# Kikao

**Description**: A Tanzanian-first event discovery, ticketing, and payments platform. Built to solve challenges such as fragmented payments, enabling event discovery, and securing entry management solutions with QR codes.

## Project Goals

Kikao is designed to:
- Provide a centralized platform for discovering Tanzanian events.
- Simplify ticket purchasing and enable mobile money payments (M-Pesa, Airtel Money, Tigo Pesa, and cards).
- Eliminate fraudulent tickets with QR-based entry management.
- Empower event organizers with a sales dashboard, guest list downloads, and ticket scanning capabilities.

### Target Audience
Built for Tanzanian promoters, creatives, corporates, churches, conferences, nightlife events, and more!
